# Music Player Card

A modern, aesthetically pleasing Music Player Card UI built with HTML and CSS. This project features a glassmorphism design, a custom audio player interface, and a native Popover API context menu.

## Features

- **Glassmorphism Design**: Sleek, modern UI with backdrop blur and transparency effects.
- **Custom Audio Controls**: Styled play/pause, next/prev buttons, and progress bar.
- **Context Menu (Popover API)**: Native HTML Popover API used for the "More Options" menu, featuring:
  - Sub-menus for Playlists and Sharing.
  - Spotlight effect on hover.
  - Smooth animations.
- **Responsive implementation**.

## Technologies Used

- **HTML5**: Semantic structure and Popover API.
- **CSS3**: Variables, Flexbox, CSS Grid, Anchor Positioning (experimental), and sophisticated animations.
- **Remix Icon**: For vector icons.

## How to Run

1.  Clone the repository:
    ```bash
    git clone https://github.com/codebysrk/MusicPlayerCard.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd MusicPlayerCard
    ```
3.  Open `index.html` in your preferred web browser.
